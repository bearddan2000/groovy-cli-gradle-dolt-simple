# groovy-cli-gradle-dolt-simple

## Description
A groovy gradle build, that connects to dolt database.

## Tech stack
- groovy
- gradle
  - 6.8.2

## Docker stack
- dolthub/dolt-sql-serverdb:latest
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
